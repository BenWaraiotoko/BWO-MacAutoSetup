# Warp Terminal Configuration

Custom themes and launch configurations for Warp terminal.

## Themes

### Catppuccin Mocha (`catppuccin_mocha.yml`)
Soothing pastel theme with excellent contrast, matching the Neovim configuration.

### Aura Theme (`aura-theme.yaml`)
Alternative dark theme with vibrant colors.

## Launch Configurations

Custom launch configurations stored in `launch_configurations/`.
